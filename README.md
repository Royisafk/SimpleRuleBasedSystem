1. Project Title
Simple Medical Expert System

2. Project Description
This project is a simple rule-based medical expert system developed using Python in a Jupyter Notebook within Visual Studio Code (VS Code). The system uses knowledge representation techniques to store medical knowledge and logical inference to diagnose possible illnesses based on symptoms entered by the user.
The expert system analyzes user-provided symptoms and applies predefined IF–THEN rules to determine the most likely disease.

3. Objectives
Represent knowledge using facts and rules.
Design a simple rule-based expert system.
Apply logical inference techniques.
Diagnose illnesses based on symptoms.
Demonstrate knowledge representation concepts in Artificial Intelligence.
Organize and document a project using GitHub.

6. Rules Applied
 Rule 1
IF Fever AND Headache AND Fatigue
THEN Malaria
 Rule 2
IF Cough AND Chest Pain AND Fatigue
THEN Pneumonia
 Rule 3
IF Sneezing AND Runny Nose AND Sore Throat
THEN Flu
 Rule 4
IF Vomiting AND Diarrhea AND Fatigue
THEN Food Poisoning

8. How to Run the Program
1. Open the project folder in VS Code.
2. Open the notebook file:
   Simple Medical Expert System.ipynb
3. Run notebook cell.
4. Enter symptoms separated by commas when prompted.
5. The system will apply the rules and display the possible illness.
Example Input:
Fever, Headache, Fatigue
Example Output:
Possible Illness:
Malaria
